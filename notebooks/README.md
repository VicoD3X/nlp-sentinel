# Notebooks

Le notebook `01_sentiment_analysis_modeling.ipynb` contient l'exploration Sentiment140 et la démarche de modélisation.

Il sert à documenter :

- la préparation d'un échantillon équilibré ;
- les essais de modèles NLP ;
- la logique de construction de l'artefact final ;
- le lien entre expérimentation et application.

L'application Streamlit et l'API utilisent l'artefact suivi dans `models/tfidf_vectorizer.pkl`.
